Welcome to the Documentation Page

This is REST API for an application called “myFlix” that interacts with a database that stores data about different movies.


Business Logic URL endpoint HTTP Method Request body data format Response body data format
Get list of all movies /movies GET None a JSON object containing data about all movies
Get data about a single movie by title /movies/[name] GET None a JSON object containing data about a single movie. Example:

{
 id: 1,
 name: 'movie_title',
 director: {
 name: 'director',
 bio: 'bio about director',
 birthyear: 1990,
 deathyear: 2022,
 }
 genre: {
 name: 'thriller',
 description: 'genre description',
 }
 description: 'Lorem ipsum',
 imageUrl: 'https://via.placeholder.com/30',
 featured: true
}
Get data about a genre by name /movies/genre/[genre name] GET None a JSON object containing data about a specific genre
Get data about a specific director of a movie /movies/[name]/director GET None a JSON object with data about a specific director
Add a user /users POST a JSON object containing data about the user to be added

Example:

{
 name: 'Bob',
}		 a JSON object containing the user that was just added
Update a user's username /users/[id] PUT None A message indicating that the username has been updated
Add movie to user's list of favorite movies /users/[id]/[movieTitle] PUT None A message confirming that the movie was added to the favorites list
Removes a movie from the users favorites list /users/[id]/[movieTitle] DELETE None A message indicating that the movie was removed from the user's list
Removes user's profile /users/[id] DELETE None A message confirming that the user was successfully deleted
